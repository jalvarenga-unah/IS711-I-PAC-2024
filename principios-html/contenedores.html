Container (main)

El elemento main es un contenedor que se utiliza para agrupar el contenido principal de un documento HTML.

El elemento main debe contener el contenido principal de un documento HTML, como artículos, secciones, encabezados, pies de página, etc.

Container (div)

Un contenedor es un elemento que agrupa a otros elementos, es decir, es un elemento que contiene a otros elementos.

Los contenedores son elementos que permiten agrupar a otros elementos, de tal forma que se puedan aplicar estilos a todos los elementos que contienen.
